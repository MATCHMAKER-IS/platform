# demos

基盤パッケージ(`@platform/*`)の使い方を示すデモアプリ置き場。`apps/` と同じ構成です。

- 目的: 各基盤機能の「呼び出し例」を示し、実アプリ実装の見本にする。
- ルール: `apps/` と同様、**基盤(`packages/`)のソースは編集しない**。基盤は公開 API 経由で使う。
- すべて自己完結(DB・外部サービス不要)。DB/メール等は基盤のメモリ実装で代替する。

## デモには 2 種類ある

| 種類 | 数 | 起動 | 使い方 |
|---|---|---|---|
| **アプリ型** | 1（`showcase`） | `pnpm dev:demos` → http://localhost:3001 | ブラウザで動かして体験する |
| **コンポーネント型** | 26 | 起動しない | **ソースを読んで実装の参考にする**。自分のアプリにコピーして使う |

コンポーネント型は `src/*.tsx` などに部品や結線ロジックだけを置いています。`dev` スクリプトは持たず、単体では起動しません(意図的にそうしています。デモごとにアプリを立てると起動ポートも保守も増えるため)。

**使い方の探し方**: `pnpm mcp:catalog` を繋いだ AI から `find_examples("請求書")` のように引くと、該当デモと使用パッケージが返ります。手で探すなら下の一覧か、各デモの README を見てください。
